## У меня установлено
 - Node.js - v10.10.0

## Startup
```
git clone https://github.com/Ajmdag/01-responsive-markup.git
cd 01-responsive-markup
npm install
gulp
```

Откроется вкладка в браузере на http://localhost:3000/
